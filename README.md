# Ankit Kumar — Portfolio

A modern, responsive personal portfolio built with React and Vite, featuring a professional tech-themed design with smooth animations.

## 🔗 Live

**[ankit-portfolio-ten-tawny.vercel.app](https://ankit-portfolio-ten-tawny.vercel.app)**

## Preview

![Portfolio Preview](https://img.shields.io/badge/Status-Live-34d399?style=for-the-badge)

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Framework | React 19 |
| Build Tool | Vite 6 |
| Animations | Framer Motion |
| Icons | React Icons |
| Styling | CSS Modules |
| Deployment | Vercel |

## Features

- **Animated Hero** — Rotating role titles with blur transitions
- **About** — Bio, stats, and highlight cards
- **Skills** — Categorized tech stack with hover effects
- **Projects** — Featured work with themed AI/ML/Systems placeholders
- **Certifications** — Auto-scrolling marquee carousel
- **Contact** — Interactive form with copy-to-clipboard, availability status
- **Responsive** — Fully mobile-friendly
- **Tech Background** — Dot-grid, hex-circuit, and aurora wave overlays
- **Professional Palette** — Softer emerald/cyan/lavender tones for readability

## Getting Started

```bash
# Clone
git clone https://github.com/Dev-anxit/Portfolio.git
cd Portfolio

# Install
npm install

# Dev server
npm run dev

# Production build
npm run build
```

## Project Structure

```
src/
├── components/
│   ├── Hero/            # Landing section with animated roles
│   ├── About/           # Bio, stats & highlights
│   ├── Skills/          # Tech stack grid
│   ├── Projects/        # Featured projects
│   ├── Certifications/  # Certificate carousel
│   ├── Contact/         # Contact form & info
│   ├── Navbar/          # Fixed navigation
│   ├── Footer/          # CTA banner & links
│   ├── MeshBackground/  # Tech grid background
│   ├── ScrollProgress/  # Scroll indicator
│   └── ScrollToTop/     # Back to top button
├── index.css            # Global theme variables
├── main.jsx             # Entry point
└── App.jsx              # Root component
```
